# Personal Portfolio Website

## Project Description

This project is a simple personal portfolio website created using HTML5 and CSS3. The website introduces me as a Computer Science student, shows my programming skills, and provides my contact information.

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
Personal-Portfolio/
│── index.html
└── README.md
```

## Website Sections

### Header
The header displays my full name and my title as a Computer Science student.

### About Me
This section contains a short introduction about myself and my interest in programming and software development.

### Programming Skills
This section shows my programming skills:

- Python
- Java
- JavaScript
- HTML5 & CSS3

### Contact Me
This section includes my GitHub profile and my email address so anyone can contact me.

### Footer
The footer contains the copyright information.

## Design

The website has a simple and clean design. I used CSS to add colors, spacing, rounded corners, shadows, and hover effects for the links. The layout is also responsive and works well on different screen sizes.

## How I Built the Project

I started by creating the HTML file and adding the main sections of the website, including the Header, About Me, Programming Skills, Contact Me, and Footer sections.

After that, I used CSS to style the website by changing the background color, text colors, spacing, rounded corners, and adding hover effects for the links.

Finally, I tested the website in a web browser to make sure all sections were displayed correctly and all links worked properly.

## How to Run

1. Download the project files.
2. Open the `index.html` file in any web browser.
3. The website will be displayed.
